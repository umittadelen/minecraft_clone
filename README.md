<h1>minecraft_clone</h1>

<p>A basic procedural voxel terrain experiment made with <strong>Ursina Engine</strong>.</p>

<h2>Features</h2>
<ul>
  <li>Procedural custom terrain generation with smoothed heights</li>
  <li>Place and remove voxels with simple mouse input</li>
  <li>Easy to modify and extend</li>
</ul>

<h2>Requirements</h2>
<ul>
  <li>Python 3.8+</li>
  <li><a href="https://www.ursinaengine.org/">Ursina Engine</a></li>
</ul>
    
<h2>Setup</h2>
<ol>
  <li>Clone the repository:
    <pre><code>git clone https://github.com/eard3n1/minecraft_clone.git</code></pre>
  </li>
  <li>Install Ursina:
    <pre><code>pip install ursina</code></pre>
  </li>
  <li>Run the program:
    <pre><code>python main.py</code></pre>
  </li>
</ol>
